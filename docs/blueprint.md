# **App Name**: E-Reader Pro

## Core Features:

- Document Display: Display PDF and EPUB documents.
- Text Annotation: Enable text highlighting and underlining with color options.
- Save Edited Book: Allow users to save documents with annotations.
- Library: Display all ebooks in the device and allow opening books from there

## Style Guidelines:

- Primary color: White or light grey (#F5F5F5) for clean reading experience.
- Secondary color: Dark grey (#333) for text to provide high contrast.
- Accent: Teal (#008080) for highlights, annotations, and buttons.
- Clean, distraction-free reading interface with customizable margins and line spacing.
- Simple, intuitive icons for annotation tools.
- Smooth transitions for page turns and tool selections.
